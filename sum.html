1. What is CSS? Why do we use it?

CSS (Cascading Style Sheets) is a language used for styling and to describe the presentation (layout, colors, fonts) of HTML documents.

2. In how many ways can we create CSS styling?

CSS styling can be added in three main ways: Inline CSS: Styles are added directly to the HTML element using the style attribute. Internal (Embedded) CSS: Styles are placed inside tag within the HTML document head tag External CSS Styles are placed in an external .css file, linked to the HTML file.

3.Difference between the 3 ways of CSS styling:

Inline CSS: Overrides all other styles but can make HTML harder to maintain. Internal CSS: Good for small websites or pages with unique styles but doesn’t allow for reusability across pages. External CSS: Best for larger projects, allows separation of concerns and reusability across multiple pages

4. What is specificity among the ways of CSS styling?

Specificity determines which CSS rule is applied by calculating the "weight" of selectors. Inline styles have the highest specificity, followed by ID selectors, class selectors, and element selectors.

5.Syntax of CSS:

selector { property: value; }

6.What is a selector?

A selector in CSS defines the HTML element(s) to which the styles apply. Examples include element, class, id selectors.

7. Types of selectors:

Element selectors: Target HTML elements like h1, p. Class selectors: Target elements with a specific class (e.g., .my-class). ID selectors: Target a single element with a specific ID (e.g., #my-id). Universal selector (*): Targets all elements. Attribute selectors, pseudo-classes, and pseudo-elements also exist.

9. Difference between visibility: hidden and display: none:

visibility: hidden: The element is invisible but still takes up space. display: none: The element is removed from the layout completely.

10. What is object-fit?

object-fit controls how content like images or videos are resized within their container. Common values are: fill, contain, cover, scale-down, none.

11. What is the background property?

The background property allows setting background styles for elements (color, image, position, etc.). Example properties: background-color background-image background-repeat background-position

12. What is the position property?

position specifies the positioning method for an element: static, relative, absolute, fixed, sticky.

13. Difference between position: sticky and position: fixed:

sticky: The element sticks to a position based on the scroll but only within its parent container. fixed: The element is fixed relative to the viewport and stays there when scrolling.

14.Difference between position: relative and position: absolute:

relative: Moves an element relative to its normal position. absolute: Positions the element relative to the nearest positioned ancestor.

15.What is overflowing in CSS?

Overflowing occurs when an element's content is too large to fit inside its box. Use the overflow property to handle it (visible, hidden, scroll, auto).

16. What is z-index in CSS?

z-index controls the stacking order of elements. Higher values bring elements to the front. Default is auto (0).

17. What is a clear property?

The clear property specifies on which sides floating elements are not allowed to float. Values: left, right, both.

18. What are CSS variables?

CSS variables (Custom Properties) allow reusability of values. Defined with --variable-name and accessed with var(--variable-name).

19. What is :root in CSS?

:root represents the top-level element (usually ) and is used to define global CSS variables.

20. What are text properties?

Properties controlling text appearance: color, font-size, font-family, text-align, text-transform, text-shadow, etc.

21. What are font properties?

Font properties include: font-family, font-size, font-weight, font-style, line-height, letter-spacing.

22. What is a Flexbox in CSS?

Flexbox is a layout system that allows easy alignment of elements along the main and cross axes. It provides flexible container management.

23. What is a grid layout system?

Grid provides a two-dimensional layout system, allowing content to be organized into rows and columns.

24. Difference between Flex and Grid:

Flexbox: One-dimensional (row or column). Grid: Two-dimensional (rows and columns).

25. What are media queries in CSS?

Media queries allow applying styles based on the device’s characteristics (like width). Useful for responsive design.

26. What are transitions in CSS?

Transitions allow smooth changes in CSS property values over a duration.

27. What are transforms in CSS?

Transforms allow 2D/3D manipulation of elements (e.g., rotate(), scale(), skew(), translate()).

28. How can you center a div?

Using display: grid; place-items: center; is one method to center a div both vertically and horizontally.

29. Difference between Transitions vs Transforms vs Animations:

Transitions: For smooth changes over time. Transforms: For element transformations like rotation, scaling. Animations: More complex, allow keyframe-based animations.

30. General Selectors:

ID, class, element, universal selector, grouping selectors (e.g., h1, h2).

31. Difference between Pseudo-class vs Pseudo-element:

Pseudo-class: Targets an element based on its state (e.g., :hover). Pseudo-element: Targets a part of an element (e.g., ::before, ::after).

32.What is a rule set?

A rule set consists of a selector and declaration block (e.g., h1 { color: red; }).

33. What is the box-model?

The CSS box model defines how elements are structured (content, padding, border, margin)

34. Box-model properties:

Content, padding, border, margin.

35. Difference between padding and margin:

Padding: Space inside the element’s border. Margin: Space outside the element’s border.

36. Difference between outline and border:

Border: Part of the element box, takes up space. Outline: Does not take up space, drawn outside the border.

37. Types of colors in CSS:

Direct color names, RGB, RGBA, hexadecimal, HSL, HSLA.

38. What is opacity?

Opacity controls the transparency of an element. Value ranges from 0 (transparent) to 1 (opaque).
